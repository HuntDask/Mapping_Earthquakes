# Mapping_Earthquakes

## Overview 
Mapping Earthquakes for a non profit company using JS and APIs to interpret and visualize earthquake data around the world. Through our creation of the HTML webpage and using GeoJSON data from our ,apping API, we are able to observe and specifcally detail many earhtquakes in all parts of the world. We can see tetonic plates running througout the map, along with different size of colored points that indicate the various levels fo the earhtquakes happening in the area.
